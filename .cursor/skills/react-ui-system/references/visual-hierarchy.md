# Visual Hierarchy

Hierarchy tot giup app nhin hien dai, ro rang, va de scan.

## Preferred Principles

- clear page title and action zone
- obvious primary action, quieter secondary actions
- strong spacing rhythm between sections
- limited emphasis colors, mostly driven by tokens
- restrained shadows and borders

## Density Guidance

- keep dense data screens readable with whitespace and alignment
- avoid stacking too many visual weights in the same area
- use muted text for supporting context, not for important actions

## Guardrails

- Do not use color as the only signal for importance.
- Do not make every card, badge, and button visually loud.
- Keep hierarchy consistent across light and dark mode.
