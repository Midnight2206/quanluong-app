# Layout Adaptation

## Common Patterns

- desktop sidebar becomes mobile drawer
- multi-column grid becomes stacked sections
- toolbar actions wrap or collapse on smaller screens
- content width stays readable instead of stretching edge-to-edge unnecessarily

## Good Rules

- keep headers flexible and wrap-friendly
- keep main content spacing smaller on mobile and roomier on desktop
- prefer sticky or persistent navigation only when the screen can support it

## Guardrails

- Avoid fixed widths that break narrow screens.
- Avoid requiring horizontal scrolling for the whole page.
