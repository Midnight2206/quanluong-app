# UI Patterns

Notify UI nen ro rang nhung khong qua on ao.

## Common Surfaces

- bell button with unread badge
- dropdown list for recent notifications
- drawer or panel for a fuller notification center

## Good Practices

- show severity visually but calmly
- keep one line of title plus short message preview
- provide clear read and navigation actions

## Guardrails

- Do not overwhelm the header with long notification text.
- Keep badge counts concise.
- Separate persistent notification UI from error fallback UI.
