# Content Density

Responsive khong chi la layout, ma con la mat do thong tin.

## Preferred Practices

- reduce visual noise on smaller screens
- keep headings compact but still clear
- use slightly tighter spacing on phone and more breathing room on desktop
- keep cards readable, not bloated

## Guardrails

- Do not shrink text too far to preserve layout.
- Do not keep desktop-level density on mobile if it hurts readability.
