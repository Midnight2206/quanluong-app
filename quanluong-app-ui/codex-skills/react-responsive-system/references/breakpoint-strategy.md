## Preferred Mindset

- start with a usable phone layout
- expand to tablet with better spacing or two-column composition
- enhance desktop with richer density and persistent navigation

## Typical Breakpoint Intent

- `sm`: small phone to large phone adjustments
- `md`: tablet and medium-width surfaces
- `lg`: desktop layout activation
- `xl`: wide dashboard refinement

## Guardrails

- Do not use breakpoints randomly.
- Each breakpoint should represent a meaningful layout decision.
