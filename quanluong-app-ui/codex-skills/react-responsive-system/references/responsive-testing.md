Responsive tot can duoc verify chu khong chi viet breakpoint roi tin la du.

## Checkpoints

- phone portrait
- large phone
- tablet width
- laptop desktop
- wide desktop

## Verify

- no clipped content
- no accidental horizontal overflow
- action buttons remain reachable
- drawers and overlays can be closed easily
- text remains readable

## Guardrails

- Test both empty and filled states.
- Test loading, error, and real-content cases at smaller widths too.
