# Empty State Patterns

Empty state la trang thai hop le, khong phai loi.

## Good Empty State Content

- one clear title
- short explanation
- optional primary action such as create, clear filters, or retry fetch

## Common Empty State Cases

- no records exist yet
- no results match active filters
- feature is available but not configured

## Guardrails

- Explain why the screen is empty when that context helps the user.
- Match the action to the empty-state cause.
- Do not overload empty states with troubleshooting text better suited for errors.
